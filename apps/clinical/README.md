# ZeroDash Clinical API

This repository details the data model and API endpoints for the clinical module in ZeroDash.

## Installation
```bash
$ npm install
```

## Running the app
Have a look at `.env.example` for the variables you'll need to run. Follow the guide [here](https://childrenscancerinstitute.atlassian.net/wiki/spaces/CSD/pages/2069233680/Doppler+Secrets+Management) for more information.
Execute the following command to run the app. If needed, add the `:dev` tag after `start` for verbose logging.
```bash
$ npm run start
```
