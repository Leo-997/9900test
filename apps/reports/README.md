# BIXENG Microservice Template #

This repository is intented to act as a template for any microservices deployed within the ZeroDash or Zero Cloud space.
Because of this, it should **NOT** be modified directly, and should instead be forked as a foundation for a new microservice.

After forking, search for all instances of `TODO` to find all the adjustments required in the code.

### Tech stack ###

* TypeScript
* NestJS (MVC architecture)
* knex (database migrations + queries)
